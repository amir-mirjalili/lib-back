## Description

Local Library Management System

---

## Installation

```bash
$ npm i
```

make a copy of .env.example and rename the copy file to .env and change the values.

## Running the app

```bash
# ts-node
$ npm run start

# development + nodemon
$ npm run dev

# production mode
$ npm run build
$ npm run prod
```

## Testing the app

```bash
# linux
$ npm run test

# windows
$ npm run test:w
```
